PUNTO SMART OS - v90 SMART SEARCH

Cambios principales:
- Nueva barra Smart Search.
- Permite buscar en Web, IA, YouTube, MercadoLibre, Gmail, Drive, Maps y Trámites.
- Mantiene la búsqueda local dentro de los accesos con el botón "Mis accesos".
- Enter o botón Buscar abre el destino seleccionado.
- Se quitaron de la interfaz las opciones visibles de Exportar/Importar JSON que confundían al usuario.

Para subir a GitHub Pages:
1. Subir todos los archivos de esta carpeta al repo plus2.
2. Reemplazar index.html, script.js, styles.css y smartos-extra.css.
3. Mantener plus-config.js y plus-drive.js.
4. Abrir la web con ?v=90 para evitar caché.
